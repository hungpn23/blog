# Nestjs boilerplate

A nestjs boilerplate project

## Tech stack

Nestjs, MySQL

## Installation

Install with pnpm

Pull mysql docker image

```bash
  docker pull mysql:9.1
```

```bash
  docker compose up -d
  git clone https://github.com/hungpn23/blog.git
  cd blog
  pnpm install
```

Create and configure `.env` file from `example.env`

```bash
  pnpm start:dev
```
